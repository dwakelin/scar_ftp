To run it:

In one terminal run python -u server.py
In another terminal run python -u client.py
From client you can enter HELP which will bring up the various commands and what they do:

	Conn- Connects to the server
	List- Lists all the files on the server
	Upld- Allows you to upload a file onto the server
	Dwld- Allows you to download a file onto the server
	Delf- Allows you to delete a file from the sever
	Quit- Closes the socket
