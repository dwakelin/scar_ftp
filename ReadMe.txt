Available commands:

Help- Displays commands
Conn- Connects to the server
List- Lists all the files on the server
Upld- Allows you to upload a file onto the server, type upld, then file name
Dwld- Allows you to download a file onto the server, type dwld, then file name
Delf- Allows you to delete a file from the sever, type delf, then file name
Quit- Closes the socket


In one terminal rum pyhton -u server.py
In another terminal run ython -u client.py
from client you can:
list (to get files) etc
